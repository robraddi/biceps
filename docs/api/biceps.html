class biceps.Preparation(scheme=None, states=0, indices=None, exp_data=None, top=None, data_dir=None, precomputed_pf=False)¶

A parent class to prepare input files for BICePs calculation.

Parameters

  • scheme (str) – type of experimental observables {‘noe’,’J’,’cs_H’,’cs_Ha’,’cs_N’,’cs_Ca’,’pf’}

  • default=0 state (int) – number of states

  • default=None indices (str) – experimental observable index (*.txt file)

  • default=None exp_data (str) – experimental measuremnets (*.txt file)

  • default=None top (str) – topology file (*.pdb)

  • default=None data_dir (str) – precomputed data directory (should have *txt file inside)

build_exp_ref_pf(rest_index)¶

Calculate the MLE average PF values for restraint j across all structures,

>> beta_PF_j = np.array(protectionfactor_distributions[j]).sum()/(len(protectionfactor_distributions[j])+1.0)

then use this information to compute the reference prior for each structures. * VAV: NOTE that this reference potential probably should NOT be used for protection factors, PF! *

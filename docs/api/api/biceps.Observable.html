biceps.Observable¶

class biceps.Observable.NMR_Chemicalshift(i, exp, model)¶

A data containter class to store a datum for NMR chemical shift information.

Initialize the derived NMR_Chemicalshift class.

Parameters

i (int) – atom indices from the conformation defining this chemical shift

Variables

  • exp – the experimental chemical shift

  • model – the model chemical shift in this structure (in ppm)

>>> biceps.Observable.NMR_Chemicalshift(i, exp, model)
class biceps.Observable.NMR_Dihedral(i, j, k, l, exp, model, equivalency_index=None, ambiguity_index=None)¶

A data containter class to store a datum for NMR dihedral information.

Initialize NMR_Dihedral container class

param int i,j,k,l

atom indices from the conformation defining this dihedral

var exp

the experimental J-coupling constant

var model

the model distance in this structure (in Angstroms)

var equivalency_index

the index of the equivalency group (i.e. a tag for equivalent H’s)

var ambiguity_index

the index of the ambiguity group

(i.e. some groups distances have distant values, but ambiguous assignments. Posterior sampling can be performed over these values)

>>> biceps.Observable.NMR_Dihedral(i, j, k, l, exp,  model)

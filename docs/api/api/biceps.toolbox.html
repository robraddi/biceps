biceps.toolbox¶

biceps.toolbox.sort_data(dataFiles)¶

Sorting the data by extension into lists. Data can be located in various directories. Provide a list of paths where the data can be found. Some examples of fileextensions: {.noe,.J,.cs_H,.cs_Ha}.

Parameters

dataFiles (list) – list of strings where the data can be found

Raises

ValueError – if the data directory does not exist

>>> biceps.toolbox.sort_data()

biceps.toolbox.get_J3_HN_HA(top, traj=None, frame=None, model='Habeck', outname=None)¶

Compute J3_HN_HA for frames in a trajectories.

Parameters

  • traj (mdtraj.Trajectory) – Trajectory

  • top (mdtraj.Topology) – topology file

  • frame (list) – specific frame for computing

  • model (str) – Karplus coefficient models [“Ruterjans1999”,”Bax2007”,”Bax1997”,”Habeck” ,”Vuister”,”Pardi”]

  • outname (str) – if not None, the output will be saved and a file name (in the format of string) is required.

biceps.toolbox.g(f, max_tau=10000, normalize=True)¶

Calculate the autocorrelaton function for a time-series f(t). INPUT f - a 1D numpy array containing the time series f(t)

PARAMETERS max_tau - the maximum autocorrelation time to consider. normalize - if True, return g(tau)/g[0]

RETURNS result - a numpy array of size (max_tau+1,) containing g(tau).

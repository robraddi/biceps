Analysis

In this tutorial, we show the user how to instantiate the biceps.Analysis class. Analysis uses MBAR to get predicted populations of each conformational states and BICePs score. We also provide a short description of the output data from Analysis and embed the figures of posterior distribution of populations & nuisance parameters. Please refer to the documentation of Analysis for more specific details.

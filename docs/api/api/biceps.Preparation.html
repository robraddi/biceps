biceps.Preparation¶

class biceps.Preparation.Preparation(scheme=None, states=0, indices=None, exp_data=None, top=None, data_dir=None, precomputed_pf=False)¶

A parent class to prepare input files for BICePs calculation.

Parameters

  • scheme (str) – type of experimental observables {‘noe’,’J’,’cs_H’,’cs_Ha’,’cs_N’,’cs_Ca’,’pf’}

  • default=0 state (int) – number of states

  • default=None indices (str) – experimental observable index (*.txt file)

  • default=None exp_data (str) – experimental measuremnets (*.txt file)

  • default=None top (str) – topology file (*.pdb)

  • default=None data_dir (str) – precomputed data directory (should have *txt file inside)

write(out_dir=None)¶

Write BICePs format input files for available experimental observables.

Parameters

default=None out_dir (str) – output directory

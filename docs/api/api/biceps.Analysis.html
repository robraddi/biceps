biceps.Analysis¶

class biceps.Analysis.Analysis(states=0, precheck=True, resultdir=None, BSdir='BS.dat', popdir='populations.dat', picfile='BICePs.pdf')¶

A class to perform analysis and plot figures.

Parameters

  • default=0 states (int) – number of conformational states

  • default=None data (str) – BICePs input data directory converted from precomputed observables

  • default=None resultdir (str) – output files directory

  • default='BS.dat' BSdir (str) – output BICePs score file name

  • default='populations.dat' popdir (str) – output BICePs reweighted populations file name

  • default='BICePs.pdf' picfile (str) – output figure name

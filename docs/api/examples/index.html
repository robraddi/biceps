Tutorials: understanding the workflow

  • Preparation - prepare input files for `biceps`
  • Restraint - constructing the ensemble using experimental restraints
  • PosterSampler - sampling of the posterior distribution of a given ensemble
  • Analysis - predicted populations of each conformational states and compute BICePs score
  • Convergence - check the convergence of MCMC sampling
  • Multiprocessing Lambdas - run biceps in a fraction of the time by parallelizing lambda values
  • Toolbox - collection of methods outside the typical `biceps` workflow

Full Examples

Each of the following examples is .

  • Cineromycin B - determine the solution-state conformational populations of a 14-membered macrolide antibiotic. This example is based on previous work (DOI: 10.1002/jcc.23738).
  • Albocycline - compute conforamtional populations of a 14-membered macrolactone. Multiprocess lambda values and sample the posterior distribution for each simultaneously. This example is based on a previously published work (DOI: 10.1016/j.bmc.2018.05.017)

You are welcome to contribute your own examples and please let us know by simply submit a pull request on our Github!

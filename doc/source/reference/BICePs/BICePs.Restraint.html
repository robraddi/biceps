The parent class of all Restraint() objects. Parameters ———- PDB_filename: string

A topology file (*.pdb)

Reference potential.

Default = np.log(1.02)
sigma_min: float
Default = 0.05
sigma_max: float
Default = 20.0
use_global_ref_sigma: bool
Default = True

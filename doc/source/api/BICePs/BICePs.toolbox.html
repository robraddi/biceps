BICePs.toolbox.get_J3_HN_HA(traj, top, frame=None, model='Habeck', outname=None)¶

Compute J3_HN_HA for frames in a trajectories. Parameters ———- traj: trajectory file top: topology file frame: specific frame for computing model: Karplus coefficient models [“Ruterjans1999”,”Bax2007”,”Bax1997”,”Habeck” ,”Vuister”,”Pardi”] outname: if not None, the output will be saved and a file name (in the format of string) is required.

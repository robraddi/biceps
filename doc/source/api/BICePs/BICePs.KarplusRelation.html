BICePs.KarplusRelation¶

class BICePs.KarplusRelation.KarplusRelation¶

A class containing formulae giving J-coupling values from dihedral angle.

Initialize the KarplusRelation class.

Allylic(angle)¶

Returns predicted J-coupling constant given an angle in degrees, for an allylic sp3-sp2 bond

Altona_HH(angle)¶

NOTES: From: http://janocchio.sourceforge.net/janocchio.docbook/ch04s03.html Altona is good if the protons are attached to two sp3 carbons. This equation takes into account the electronegativity of all adjacent atoms.

Haasnoot, C., de Leeuw, FA, Altona, C (1980). Tetrahedron 36(19), 2783-2792

Aydin_HC(angle)¶

Returns predicted J-coupling constant given an angle in degrees. NOTES: Mag. Res. Chem. (1990) 28, 448-457 See also Janocchio paper

BothnerBy_HH(angle)¶

Returns predicted J-coupling constant given an angle in degrees. NOTES: good for cycloalkanes. Bothner-By, 1965 (14)

J(angle, key)¶

Returns the predicted J-coupling constant given an angle in degrees, and a key ‘Karplus_HH’, e.g.>

Karplus2_HH(angle)¶

Returns predicted J-coupling constant given an angle in degrees. NOTES: A simple Karplus relation from the Janocchio documentation.

Magnetic Resonance in Chemistry, 45(7), 595-600. doi:10.1002/mrc/2016

Karplus_HH(angle)¶

Returns predicted J-coupling constant given an angle in degrees.

Karplus_HH_appO(angle)¶

Returns predicted J-coupling constant given an angle in degrees, for sp3-single bond HH where there is an antiperiplanar Oxygen

Tvaroska_HC(angle)¶

Returns predicted J-coupling constant given an angle in degrees. NOTES: Adv. Carbohydrate Chem. Biochem. (1995) 51, 15-61 See also Janocchio paper

Wasylichen_HC(angle)¶

Returns predicted J-coupling constant given an angle in degrees. NOTES: Recommended for general use (Can J Chem (1973) 51 961.) See also Janocchio paper

BICePs.Preparation¶

  • Classes
class BICePs.Preparation.Preparation(scheme=None, states=0.0, indices=None, exp_data=None, top=None, data_dir=None)¶

A parent class to prepare input files for BICePs calculation

Prepare BICePs input files (converting from raw data)

scheme: {‘noe’,’J’,’cs_H’,’cs_Ha’,’cs_N’,’cs_Ca’,’pf’} states: number of states indices: experimental observable index (.txt file) exp_data: experimental measuremnets (.txt file) top: topology file (.gro, pdb, etc.) data_dir: data directory (should have txt file inside) Karplus: Karplus relation for J_coupling constants

write(out_dir=None)¶

Writes prepared files to output directory.

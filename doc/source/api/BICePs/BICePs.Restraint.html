class BICePs.Restraint.Restraint(PDB_filename, ref, dlogsigma=0.01980262729617973, sigma_min=0.05, sigma_max=20.0, use_global_ref_sigma=True)¶

The parent class of all Restraint() objects.

Initialize the Restraint class.

PDB_filename A topology file (*.pdb) data input data for BICePs (both model and exp) ref Reference potential.

add_restraint(restraint)¶

Add a new restraint data container (e.g. NMR_Chemicalshift()) to the list.

compute_neglog_exp_ref()¶

Uses the stored beta information (calculated across all structures) to calculate -log P_ref(observable[j]) for each observable j.

compute_neglog_gaussian_ref()¶

An alternative option for reference potential based on Gaussian distribution. (Ignoring constant terms)

compute_sse(debug=False)¶

Returns the (weighted) sum of squared errors for chemical shift values

Load in the experimental chemical shift restraints from a known file format.
